# ШТУРМ (SHTURM)

Аркадный run-and-gun платформер в стиле классических 16-битных шутеров. Python + pygame, веб-сборка через pygbag, цель — VK Mini Apps.

Проект изначально назывался "Contra-VK" — переименован в ШТУРМ (SHTURM), чтобы избежать пересечения с торговой маркой Konami.

## Что есть (v0.15)

### ✨ v0.15 — Камикадзе-дрон: пикирующий смертник

- **Новый враг: Kamikaze** — летающий дрон-смертник с боеголовкой. Кружит в воздухе и медленно дрейфует к игроку, затем **захватывает цель** (мигающий красный прицел-телеграф ~0.5 с) и **пикирует** по зафиксированному вектору, **детонируя** при касании игрока или ударе о землю. Взрыв бьёт по площади (радиус 66px).
- **Хрупкий, но опасный** — всего 1HP: можно **сбить с дистанции** одним выстрелом (безопасный контр), пока не спикировал. Это поощряет реакцию и стрельбу на упреждение.
- **Синергия с рывком (v0.14)** — если дрон уже пикирует, можно **проскочить сквозь взрыв в дэше** (кадры неуязвимости) — второй честный способ контрить атаку.
- **Телеграф и визуал** — в режиме захвата над дроном пульсирует прицельное кольцо и мигает боевая лампа; при пикировании тянется огненный след. Крылатый металлический корпус с боеголовкой-носом.
- **Где встречается** — впервые в Stage 3 (BASE, ×2) как знакомство, затем массово в Stage 4 (SKY, ×5) — небо становится по-настоящему враждебным. 300 очков.
- Теперь **12 типов врагов**.
- Bundle: **161 KB**

### ✨ v0.14 — Рывок (Dash): уворот с неуязвимостью

- **Новый приём — рывок/перекат** (клавиша **C** или **Shift**, на телефоне — голубая кнопка **DASH**): боец делает короткий стремительный бросок в сторону взгляда (~2.5× быстрее обычного бега).
- **Кадры неуязвимости (i-frames)** — во время рывка боец **проходит сквозь пули, тараны и взрывы** без урона. Это превращает дэш в навыковый инструмент: вовремя увернуться от снайперского выстрела, проскочить через очередь дрона, выйти из-под минометного снаряда.
- **Кулдаун** — после рывка пауза (~0.65 с), нельзя спамить; рывок недоступен в приседе. Это удерживает приём как тактическое решение, а не постоянный режим передвижения.
- **Визуал и звук** — голубой шлейф-послесвечение тянется за бойцом, из-под ног вырывается облачко пыли, играет свистящий «вжух».
- **Подсказка-туториал** — после первого выстрела над игроком всплывает «C — DASH (dodge!)».
- Bundle: **158 KB**

### ✨ v0.13 — Погода и атмосфера: дождь, гроза, капель, тревога

- **У каждого уровня — своя погода**, которая оживляет фон и задаёт настроение:
  - **JUNGLE (Stage 1) — дождь с грозой**: косые струи дождя по всему экрану + редкие вспышки молний, на миг подсвечивающие сцену.
  - **CAVES (Stage 2) — капель**: со свода пещеры срываются капли, падают и разбиваются о землю брызгами.
  - **BASE (Stage 3) — тревога**: пульсирующая красная виньетка по краям экрана — «сирена» вражеской базы.
  - **SKY (Stage 4) — облака**: полупрозрачные облака неспешно дрейфуют над закатным небом.
- Погода — **чисто визуальный слой**: не влияет на геймплей и коллизии, рисуется поверх кадра уже после тряски экрана, поэтому никаких артефактов по краям при шейке.
- Эффект лёгкий по производительности (предраспределённые частицы, без аллокаций в кадре).
- Bundle: **155 KB**

### ✨ v0.12 — Разрушаемое окружение: ящики и взрывбочки

- **Деревянные ящики (Crate)** — стоят на уровнях, простреливаются с одного выстрела и **гарантированно дают пикап**: чаще всего оружие, реже самоцвет или доп. жизнь. Это новый надёжный источник снаряжения — есть смысл исследовать карту и тратить патроны на ящики.
- **Взрывные бочки (Barrel)** — при попадании **детонируют**: радиальный взрыв (радиус 96px) бьёт по всем врагам рядом (урон 3) — можно зачистить целую группу одним выстрелом. Бочки **детонируют друг друга цепной реакцией**.
- **Тактический риск** — взрыв бочки задевает и игрока, если стоять вплотную (есть «безопасный зазор» — стреляй с дистанции). Бочки специально расставлены у плотных групп врагов (особенно в финале Stage 3) для эффектных цепных подрывов.
- Ящики и бочки — это **декорации-разрушаемки**: они не стреляют и не наносят урон при касании (можно пройти насквозь), ломаются только пулями. Разбросаны по всем 4 уровням (12 ящиков + 9 бочек).
- HP-бар над врагами теперь корректно масштабируется со сложностью.
- Bundle: **152 KB**

### ✨ v0.11 — Новый враг: Burrower (крот-подрывник)

- **Новый враг: Burrower** — крот, который роет под землёй и **неуязвим, пока в норе** (пустой хитбокс — пули и таран проходят сквозь). Под землёй преследует игрока по горизонтали, затем выныривает, делает прицельный выстрел и снова зарывается.
- **Телеграф появления** — пока крот под землёй, на поверхности виден земляной холмик; перед выныриванием он начинает трястись — есть время отойти.
- Уязвим только на поверхности (2HP) — нужно ловить момент. Расставлены в Stage 2 (CAVES, ×2) и Stage 3 (BASE, ×3). 350 очков.
- Теперь **11 типов врагов**.
- Bundle: **148 KB**

## Запуск локально

```
pip install -r requirements.txt
python main.py
```

Откроется окно 800×480.

## Управление

| Клавиша | Действие |
|---|---|
| `←` / `→` или `A`/`D` | Бежать |
| `↓` | Присесть |
| `↑` | Целиться вверх (можно по диагонали с `←`/`→`) |
| `Z` | Прыжок (отпустишь во время восхождения — короткий прыжок) |
| `X` | Стрелять (по 8 направлениям с учётом аим-клавиш) |
| `C` / `Shift` | Рывок-уворот (i-frames: сквозь пули и взрывы; есть кулдаун) |
| `Esc` / `P` | Пауза (в паузе — возобновить); `Q` в паузе — выход в меню |
| `M` | Mute / unmute (сохраняется) |
| `F` | CRT scanlines on/off (сохраняется) |
| `←` / `→` в меню | Сменить сложность EASY/NORMAL/HARD |
| `Enter` / `Space` | Старт / перезапуск из экрана меню/смерти/победы |

В воздухе можно стрелять вниз (`↓` + `X`) — классический контра-приём.

## Сборка веб-версии (pygbag)

```
python -m pygbag --build main.py
```

Результат — папка `build/web/` (≈60KB):
- `index.html` — точка входа
- `contra-vk.apk` / `contra-vk.tar.gz` — упакованный код
- `favicon.png`

При деплое на хостинг файлы из `build/web/` нужно класть в корень домена. Они тянут Python WASM runtime и pygame с публичного CDN `https://pygame-web.github.io/cdn/0.9.3/`.

### Локальный тест веб-версии

Простой `python -m http.server` НЕ подойдёт — pygbag-build пытается тянуть `pygame_ce` wheel с того же хоста по пути `/cdn/cp312/`, чего у статического сервера нет. Запускай pygbag-сервер, который проксирует CDN:

```
python -m pygbag --port 8001 main.py
```

Откроется `http://localhost:8001/`. Первая загрузка ~10–30 сек (тянется ~10MB WASM).

## Структура

```
contra-vk/                  # имя папки осталось contra-vk, чтобы не ломать пути
├── main.py                 # Game class: state machine (menu/intro/play/clear/over/win), LevelManager
├── game/
│   ├── constants.py        # размеры, скорости, цвета, weapon constants
│   ├── leveldef.py         # @dataclass LevelDef + Theme; ALL_LEVELS = [STAGE_1, STAGE_2, STAGE_3]
│   ├── level.py            # Level (рендер на основе theme, потолок для пещер, fog), Camera, MovablePlatform
│   ├── player.py           # Player: движение, прыжок, прицел, стрельба, respawn, draw с анимацией
│   ├── entities.py         # Bullet, MortarShell, Soldier, Turret, Drone, Jumper, Mortar,
│   │                       #   Boss, MechBoss, Pickup, Particle, Checkpoint
│   ├── sounds.py           # SoundBank — синтез 8-bit бипов в памяти, 10 эффектов
│   ├── music.py            # MusicPlayer — синтез chiptune-петель (6 треков)
│   ├── persistence.py      # cross-platform localStorage / JSON-файл для hi-score
│   └── controls.py         # TouchControls (виртуальный D-pad/JUMP/FIRE), CombinedKeys
├── assets/                 # пока пусто; всё процедурное
├── build/web/              # pygbag --build output (≈127KB)
├── assets/sprites/         # ✨ v0.8: 40 Kenney CC0 спрайтов (player, враги, тайлы)
├── assets/fonts/           # ✨ v0.8: Press Start 2P (SIL OFL)
├── tools/make_icon.py      # генератор 256×256 иконки для VK
├── tools/build_sprite_pack.py # копирует нужные Kenney файлы в assets/sprites/
├── DEPLOY.md               # инструкция GitHub Pages + VK Mini App
├── requirements.txt
├── README.md
├── VK_PUBLISH.md
└── _smoke_test.py          # headless проверка: 3 уровня, transitions, бой, new features
```
