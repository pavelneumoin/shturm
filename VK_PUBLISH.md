# Публикация ШТУРМ в VK

VK Games принимает HTML5-игры. Pygbag-сборка — это HTML+WebAssembly, подходит.

## Варианты площадок ВК

1. **VK Mini Apps** (https://vk.com/services) — встраиваемые в ВК веб-приложения. Сейчас основное место для casual-игр от индии.
2. **VK Play / vk.com/games** — большая каталог-площадка, требует партнёрского статуса для публикации; для индии практично только через Mini Apps.

Для соло-разработчика ⇒ **Mini App**.

## Шаги для VK Mini App

### 1. Захостить статичные файлы

Pygbag-сборка должна лежать на HTTPS-хостинге, доступном из ВК. Бесплатные варианты:

- **GitHub Pages** — самый простой
- **Vercel / Netlify** — автоматический деплой
- **itch.io** — может встраиваться, есть свой формат, pygbag сразу под него умеет (build/web/contra-vk.apk)

Самый дешёвый путь — GitHub Pages:

```
# создай новый репо contra-vk на GitHub
git init
git add .
git commit -m "init"
git remote add origin https://github.com/<username>/contra-vk.git
git push -u origin main

# создай ветку gh-pages с содержимым build/web/
git checkout --orphan gh-pages
git rm -rf .
cp -r build/web/* .
git add .
git commit -m "deploy"
git push origin gh-pages
```

В Settings репо включи Pages из ветки `gh-pages`. Получишь URL вида `https://<username>.github.io/contra-vk/`.

### 2. Создать Mini App в ВК

1. Зайти на https://vk.com/editapp?act=create
2. Тип — **Веб-приложение** (Mini App)
3. Название: «ШТУРМ» (или любое другое — игра уже переименована из Contra)
4. Заполнить описание, иконку (256×256, скриншот меню сойдёт)
5. В разделе «Настройки» → «Адрес для встраивания» вставить URL с GitHub Pages
6. **Embedded** = HTTPS-iframe; включить «Развернуть на весь экран»

### 3. Адаптировать под мобильные

✅ **СДЕЛАНО.** Виртуальное управление подключено в [game/controls.py](game/controls.py):
- Левый D-pad (10 направлений с диагоналями) → K_LEFT/RIGHT/UP/DOWN
- Кнопка JUMP (правая, синяя) → K_z
- Кнопка FIRE (правая, красная) → K_x
- Multi-touch через `pygame.FINGERDOWN/UP/MOTION` (можно держать D-pad одной рукой и стрелять другой)
- На десктопе тот же интерфейс работает мышью (зажимаешь ЛКМ на D-pad / на кнопке)

Оверлей рисуется только после первого касания, чтобы на десктопе с клавы он не мешал.

### 4. VK Bridge (опционально)

Если хочется хайскоры через ВК — подключи [VK Bridge JS SDK](https://dev.vk.com/ru/bridge/overview) и пробрось в pygbag через `platform.window`.

```python
# в main.py, при инициализации
import platform
score = await platform.window.vkBridge.send('VKWebAppStorageGet', {'keys': ['highscore']})
```

(работает только когда страница реально внутри ВК-фрейма)

### 5. Модерация

ВК ревьюит Mini App перед публикацией:
- Никакого порно, политики, нарушения авторских прав (название «Contra» — товарный знак Konami, переименуй на всякий случай)
- Должно работать в мобильном webview ВК (≈ Chrome 100+)
- Никакого crypto / казино
- Быть доступным без регистрации (это и так так)

Стандартный срок ревью — 1–7 дней. Чаще всего отказывают за «неработающий functionality на мобильном» — поэтому пункт 3 (тачи) важнее всего остального.

## Альтернатива — itch.io

Если ВК-модерация капризничает, можно сначала выкатить на https://itch.io/. Pygbag прямо умеет:

```
python -m pygbag --build main.py
# upload build/web/contra-vk.apk на itch.io как HTML5
```

Itch.io поддерживает формат и сразу даёт играбельную страницу. ВК потом можно вернуть.
